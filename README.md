# 원티드 프리온보딩 프론트엔드 코스 선발 과제 (할 일 목록 앱)

원티드 프리온보딩 프론트엔드 코스 선발 과제 제출용 레파지토리입니다

## 주요 기능

- 회원가입 / 로그인 기능
- 할 일 목록 추가, 삭제, 수정 기능

## 프로젝트 실행 방법

```
yarn install
yarn start
```

프로젝트 디렉토리에서 상위 스크립트 실행 후 개발 환경에서 프로젝트를 확인할 수 있습니다.

[http://localhost:3000](http://localhost:3000) 브라우저 로컬 URL에서 확인 가능합니다.
